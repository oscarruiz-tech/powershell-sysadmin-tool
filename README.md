\# PowerShell SysAdmin Tool 🚀



A professional CLI automation tool developed to monitor system health, optimize memory usage, and manage local storage efficiently.



\## 🛠 Features

\- \*\*System Monitoring:\*\* Quick hardware and OS summary.

\- \*\*Resource Management:\*\* Real-time tracking of top RAM-consuming processes.

\- \*\*Storage Optimization:\*\* Deep scan for local files >500MB (filtered to exclude cloud storage).

\- \*\*Process Recovery:\*\* One-click restart for Windows Explorer to clear UI cache and reclaim memory.

\- \*\*Security Audit:\*\* Detailed inspection of loaded DLLs and modules within the system shell.
